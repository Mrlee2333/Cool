# Cool Movie Site

一个简洁高效的影视站点前端，基于 **Vue 3 + Bulma + ArtPlayer**，高颜值，适配移动端/PC，支持多源视频聚合、剧集排序、历史播放、密码加密访问。

## ✨ 功能特性

- 多源聚合详情页，剧集列表智能折叠，去播放广告（待测试）
- 支持正序/倒序排序，加载更多剧集
- 高颜值界面，流畅体验
- 视频播放支持 ArtPlayer & HLS
- 播放进度自动保存，断点续播
- 历史记录/最近播放浮窗快捷跳转
- 移动端全屏自动横屏
- 支持密码加密访问（可设置有效期）
- Vercel 一键部署
